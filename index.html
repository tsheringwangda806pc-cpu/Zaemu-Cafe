☁️ Cloud Sync (Firebase)

Without this, orders placed on a customer's phone are only saved on their device and never reach your admin dashboard.
Connect Firebase Firestore so every order, booking, and profile syncs instantly across all devices — for free.

⚪ Local mode — configure Firebase below to sync across all devices
📋 How to get your Firebase config (takes ~5 minutes)
  1. Go to console.firebase.google.com and sign in with your Google account
  2. Click "Add project" → name it zaemu-cafe → disable Google Analytics → Create
  3. In the left sidebar click Firestore Database → Create database
  4. Choose Start in test mode (allows all reads/writes for 30 days — we'll tighten later) → Next → Enable
  5. In the left sidebar click the ⚙️ gear → Project settings
  6. Scroll to "Your apps" → click the </> Web button → register app name zaemu-web
  7. Copy the firebaseConfig values shown and paste each field below
Config is stored in your browser — paste the same config on every admin device.
